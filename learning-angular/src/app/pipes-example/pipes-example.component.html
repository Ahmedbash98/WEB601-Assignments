Pipes examples

{{ "HELLO" | lowercase }}
{{ someStringValue | uppercase }}
Without the pipe
{{ today}}
With the default date pipe
{{ today | date }}
With the medium date format
{{ today | date: "medium" }}
With the custom date format
{{ today | date: "dd/MM/yy" }}
With the medium date format AND everything will be in lowercase letters
{{ today | lowercase | date: "medium" }}
